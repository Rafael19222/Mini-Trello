# Mini-Trello
Mini Trello - Task Management Application
A lightweight task management application inspired by Trello, built with Alpine.js and Tailwind CSS.
Features

Create tasks with title and description
Search tasks by title
Filter tasks by status (To Do, In Progress, Completed)
Edit and delete tasks
Real-time statistics counter
Persistent local storage
Responsive design

Usage
Add a task: Enter a title and optional description, then click "Ajouter"
Search: Type in the search bar to filter tasks
Filter: Select a status from the dropdown menu
Edit: Click "Modifier" to update task details
Delete: Click "Supprimer" to remove a task
Change status: Use the dropdown on each task card
